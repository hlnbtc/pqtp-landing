2. Quais dados coletamos

Este site coleta dados por dois meios:

  • Pixel Meta (Facebook/Instagram): ao visitar esta página, o Pixel registra informações de navegação — como páginas visitadas, ações realizadas e dados do dispositivo — e as associa ao seu perfil nas plataformas Meta, caso você esteja conectado. Essa coleta é automática e ocorre independentemente de qualquer interação com formulários.
  • Plataforma Kiwify (checkout): ao realizar uma compra, você fornece dados diretamente à Kiwify — nome, e-mail, CPF e dados de pagamento. Esses dados são tratados pela Kiwify conforme a política de privacidade própria da plataforma, disponível em kiwify.com.br.

3. Para que usamos os dados

  • Veiculação de anúncios pagos nas plataformas Meta (Facebook e Instagram)
  • Mensuração de desempenho das campanhas (conversões, custo por aquisição)
  • Criação de públicos personalizados e semelhantes para otimização de anúncios
  • Processamento e entrega do produto adquirido (via Kiwify)

4. Base legal (LGPD)

O tratamento de dados realizado por esta página tem como base legal o legítimo interesse do responsável para fins de marketing e análise de desempenho (art. 7º, IX, Lei 13.709/2018), e a execução de contrato para os dados fornecidos no momento da compra (art. 7º, V).

5. Compartilhamento de dados

Os dados coletados via Pixel Meta são transmitidos diretamente às plataformas da Meta Platforms, Inc., conforme os termos de uso e a política de dados desta empresa. Os dados de compra são tratados pela Kiwify. Não vendemos nem cedemos dados pessoais a terceiros fora dessas relações.

6. Por quanto tempo guardamos

Os dados coletados via Pixel são retidos conforme as políticas da Meta. Os dados de compra são retidos pela Kiwify conforme as obrigações legais aplicáveis. Não mantemos base própria de dados pessoais neste momento.

7. Seus direitos

Nos termos da LGPD, você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar os dados que possuímos sobre você
  • Solicitar correção de dados incompletos ou incorretos
  • Solicitar eliminação dos dados tratados com base em consentimento
  • Revogar consentimento a qualquer momento

Para exercer qualquer desses direitos, entre em contato pelo e-mail contato@paraquemtempressa.com.br.

8. Cookies

Esta página utiliza cookies técnicos necessários ao funcionamento e cookies de rastreamento associados ao Pixel Meta. Ao continuar navegando, você concorda com esse uso. Você pode gerenciar ou bloquear cookies nas configurações do seu navegador, o que pode impactar o funcionamento de algumas funcionalidades da página.

9. Alterações nesta política

Esta política pode ser atualizada a qualquer momento. A data de última atualização constante no topo do documento indica a versão vigente. O uso continuado do site após alterações implica concordância com os novos termos.
